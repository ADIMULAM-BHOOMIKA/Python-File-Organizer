# Python-File-Organizer
Python File Handling & Automation

This project is a simple Python File Organizer built using Python in Google Colab. The program automatically organizes files into separate folders based on their file types using file handling and automation concepts.

Technologies Used

- Python
- Google Colab
- GitHub

Features

- Automatic file organization
- File reading and writing
- Folder creation and management
- Exception handling using try-except
- Log file generation

Modules Used

- os
- shutil

Supported File Types

- TXT Files
- CSV Files
- PDF Files
- JPG/PNG Images

Output

The program automatically moves files into categorized folders such as:

- TextFiles
- CSV_Files
- PDFs
- Images
